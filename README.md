# Food Billing System in C

## Description
This program is a simple food billing system written in C. It allows users to order food items (burger or pizza), specify the quantity, and calculate the total cost of their order. Users can also place multiple orders in one session.

## Features
- Display menu options with prices.
- Order food items by entering their corresponding menu numbers.
- Calculate the total amount based on the item price and quantity.
- Option to place additional orders in the same session.

## How to Use
1. Compile and run the program using a C compiler (e.g., GCC).
2. Follow the prompts to:
   - Select your order by entering the menu number.
   - Specify the quantity.
   - View the total cost of your order.
   - Decide whether to place another order or exit.
3. The program repeats until you choose to stop ordering.

## Requirements
- C Compiler (e.g., GCC or Turbo C)
